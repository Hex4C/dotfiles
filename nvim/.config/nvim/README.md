# Neovim config

My neovim config originally based on kickstart.nvim but currently heavily modified and modularized

## Installation

External Requirements:

- Basic utils: `git`, `make`, `unzip`, C Compiler (`gcc`)
- [ripgrep](https://github.com/BurntSushi/ripgrep#installation)
- Clipboard tool (xclip/xsel/win32yank or other depending on platform)
- Use the JetBrainsMonoNL NF font
- All of these tools will be checked in the health.lua check
